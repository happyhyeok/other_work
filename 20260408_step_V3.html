Part 1. 기획안 및 PPT 자동 구성

💡 NotebookLM이란?

구글에서 개발한 '나만의 AI 맞춤형 연구 보조 도구'입니다. 사용자가 직접 업로드한 문서나 링크(PDF, 텍스트, 유튜브 등)만을 기반으로 답변을 생성합니다.

다른 AI(ChatGPT 등)와 무엇이 다른가요? 일반적인 AI는 방대한 인터넷 데이터를 바탕으로 답변하여 가끔 거짓 정보(할루시네이션)를 만들어내지만, NotebookLM은 내가 제공한 자료 안에서만 답변을 찾고 요약하므로 정보의 출처가 명확하고 실무 신뢰도가 매우 높습니다.

실습 1 슈퍼 프롬프트 생성 엔진: 전문가 노트북 만들기

🎯 목표: 어떤 주제든 고퀄리티 결과물을 뽑아낼 수 있는 '프롬프트 생성 전문가' AI를 구축합니다.

1

프롬프트 제작을 위한 소스 수집

  • 새 노트북 생성: NotebookLM에서 '프롬프트 엔지니어링 전문가'라는 이름으로 새 노트북 만들기
  • 전문 자료 추가: '프롬프트 잘 쓰는 법', '효과적인 프롬프트 구조(Persona, Context, Task 등)' 등의 키워드로 자료 검색 및 5개 이상 추가
2

소스 정제 및 신뢰도 검증

  • 불필요한 소스 제거: 채팅창에 아래 프롬프트 입력하여 부실한 소스 평가 및 확인
    프롬프트
    이 소스들을 바탕으로 [도서관 행사 기획안 및 PPT 제작 전문가] 역할을 수행할 수 있는 가장 정교한 '슈퍼 프롬프트(System Instruction)'를 작성해줘
  • 복사 및 보관: 생성된 프롬프트를 따로 메모장에 복사 (실습 2에서 사용할 핵심 도구)

실습 2 여름방학 도서관 행사 기획 및 PPT 구성

🎯 목표: 실습 1에서 만든 슈퍼 프롬프트를 활용하여 실제 행사 기획안과 발표 자료 완성

🔧 사전 준비: BananaNL 확장프로그램 설치

PPT 디자인을 한차원 업그레이드시켜주는 도구

BananaNL 설치 페이지 바로가기
1

행사 기획용 노트북 구축

  • 새 노트북 생성: '여름방학 도서관 행사 기획'이라는 이름으로 새로운 노트북 만들기
  • 기획 소스 추가: 작년 행사 자료, 타 도서관 우수 사례, 여름방학 트렌드, 대상 연령층(초등생 등) 선호도 자료 업로드
2

슈퍼 프롬프트 적용

  • 맞춤형 설정: 설정 -> 맞춤형 프롬프트 란에 실습 1에서 복사해둔 슈퍼 프롬프트 붙여넣기
3

논리적 목차 및 가이드 구성

  • 세부 목차 확정: 채팅창에 아래와 같이 구체적인 슬라이드 목차 요구
    프롬프트
    여름방학 도서관 행사 기획을 위한 10장 분량의 슬라이드 목차를 짜줘. 각 장에는 구체적인 인사이트와 배치될 이미지 컨셉이 포함되어야 해
  • 소스로 변환: 확정된 목차를 메모로 저장한 뒤, 소스로 변환하여 왼쪽 리스트에 +슬라이드 가이드로 등록
4

비주얼 스타일 및 슬라이드 생성

  • 스타일 지정: 프롬프트를 직접 입력하는 대신, 화면 상단에 추가된 BananaNL 버튼을 클릭하여 제공되는 목록에서 적당한 PPT 디자인 테마를 선택합니다.
  • 최종 생성: 지침을 엄수하여 내용을 생성하도록 채팅창에 명령
    프롬프트
    등록된 +슬라이드 가이드를 엄격히 준수하여 10장의 슬라이드를 생성해줘
5

최종 리터칭

  • 슬라이드 수정: NotebookLM의 수정 기능을 이용해 특정 페이지의 제목 변경 및 캐릭터 표정 변경
  • PPT 마무리: 완성된 이미지를 파워포인트로 옮겨 학교/기관 로고 삽입 및 최종 보고서 완성

💡 실습 팁

  • 실습 1의 목적은 '도구(프롬프트)'를 만드는 것이고, 실습 2의 목적은 그 도구를 사용해 '결과물(기획안)'을 만드는 것입니다.
  • 슈퍼 프롬프트는 AI가 이해하기 쉬운 구조적 언어로 되어 있으므로, 사람이 읽기에 너무 길더라도 수정하지 않고 그대로 사용하는 것이 가장 효과적입니다.

Part 2. AI로 한글문서 변환하기

💡 스킬(Skill)과 HWPX의 이해

스킬(Skill)이란? AI가 단순히 채팅만 하는 것을 넘어, 파일 변환이나 데이터 계산 등 특정한 동작을 직접 수행할 수 있도록 만들어진 맞춤형 자동화 도구(확장 기능)입니다.
HWP vs HWPX
  • HWP: 과거 방식의 바이너리 문서로, AI가 내부 글자를 읽고 쓰기 어렵습니다.
  • HWPX: 개방형 표준 XML 형식으로, 문서 내부가 텍스트 코드로 이루어져 있어 기계가 쉽게 다룰 수 있습니다.
왜 HWPX를 쓰나요? HWPX는 압축을 풀면 문서의 구조가 코드 형태로 보입니다. AI가 이 문서를 직접 열어 {{자리 표시자}}를 찾고, 엑셀의 데이터로 정확히 바꿔치기하여 문서를 자동 완성할 수 있기 때문입니다.
1단계: 스킬 파일 및 양식 준비

3. 스킬 파일 다운로드 및 압축 해제

  • 아래 스킬 다운로드 버튼을 클릭하여 00_HWPX_Skills.zip 파일 다운로드
    스킬 다운로드 (편집자P유튜버 제공)
  • 다운로드한 파일 압축 해제 후 내부 폴더 구조 확인

4. 한글(HWPX) 문서 템플릿 제작

  • 자동화할 한글 문서 파일을 열어 데이터 입력 위치에 자리 표시자(Placeholder) 입력 (예: {{성명}}, {{주소}}, {{생년월일}})
  • 반드시 파일 > 다른 이름으로 저장 선택 후 파일 형식을 HWPX(한글 표준 문서)로 저장

5. 스킬 내 템플릿 교체 및 재압축

  • 압축 해제한 폴더 내 assets 폴더로 이동
  • 기존 템플릿 파일명 복사 후, 내가 만든 문서 파일명을 동일하게 변경
  • 변경 완료된 문서 파일을 assets 폴더에 덮어쓰기
  • 수정 완료된 전체 폴더를 다시 .zip 형식으로 압축
2단계: 클로드 환경 설정

6. HWPX 스킬 등록

  • 클로드 좌측 하단 [계정명] > [Settings] > [Features] > [Skills] 메뉴로 이동
  • Upload Skill 버튼 클릭하여 위에서 재압축한 .zip 파일 업로드
  • 스킬 목록에 'HWPX 스킬' 정상 등록 여부 확인
3단계: 문서 생성 실습

7. 데이터(엑셀) 업로드 및 명령

  • 새 채팅창을 열고, 문서에 입력할 데이터가 포함된 엑셀 파일 업로드
    연습용 엑셀파일 다운로드
  • 채팅창에 아래 프롬프트 입력
    프롬프트
    업로드한 엑셀 파일의 데이터를 바탕으로, 내장된 HWPX 스킬 템플릿을 사용하여 한글 문서를 만들어줘. 엑셀의 '이름'은 {{이름}} 위치에, '주소'는 {{주소}} 위치에 {{기관명}}위치에는 '강릉교육문화관'text를 {{기관주소}}위치에는 '강원특별자치도 강릉시 노암등길 37-1 (노암동)'text를 대입해줘.

8. 결과물 확인 및 다운로드

  • 클로드가 스킬을 실행하여 문서 생성을 완료할 때까지 대기
  • 생성 완료 후 제공되는 다운로드 링크를 클릭하여 결과물(HWPX) 저장
  • 파일을 열어 엑셀 데이터가 올바른 위치에 입력되었는지 확인

⚠️ 강사 강조 사항 (학생들이 가장 많이 실수하는 부분)

실습 중 오류가 발생한다면 다음 두 가지를 반드시 점검하도록 지도해 주세요!

  • 파일 확장자 확인: 기존 HWP 형식이 아닌 반드시 HWPX 확장자로 저장해야 스킬이 정상 작동합니다.
  • 파일명 일치 여부: 압축 파일 내 assets 폴더에 넣은 템플릿 파일명이 원본 템플릿의 파일명과 완벽히 동일한지 확인해야 합니다.
